# Modelo matemático – Sistema de mezcla

El sistema corresponde a una operación unitaria de mezcla en estado estacionario.

Se asume:
- No hay reacción
- No hay pérdidas
- Mezcla perfecta

Balance total:
F_out = F1 + F2

Balance de soluto:
m_soluto = F1·x1 + F2·x2

Fracción másica:
x_out = m_soluto / F_out
