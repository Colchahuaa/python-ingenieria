
# Análisis de Datos Industriales
Ingeniero Civil Químico | Python · Análisis Operacional · Optimización de Procesos

## Descripción
Proyecto orientado al análisis de datos operacionales provenientes de un proceso térmico industrial continuo.
Se desarrollan herramientas para limpieza de datos, cálculo de indicadores térmicos (KPIs), análisis estadístico y visualización técnica.

## Objetivos
- Evaluar estabilidad operacional
- Detectar desviaciones térmicas
- Calcular eficiencia energética
- Generar indicadores técnicos de desempeño
- Apoyar decisiones de mejora operacional

## Estructura
data/ → Datos del proceso 
src/ → Módulos de análisis 
main.py → Ejecución principal 

## Herramientas
- Python 3
- pandas
- numpy
- matplotlib
