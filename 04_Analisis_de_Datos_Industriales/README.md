
# Análisis de Datos Industriales
Ingeniero Civil Químico | Python · Análisis Operacional · Optimización de Procesos

## Descripción
Este proyecto desarrolla un enfoque estructurado para el análisis de datos operacionales provenientes de un proceso térmico industrial continuo, integrando fundamentos de ingeniería química con herramientas avanzadas de análisis en Python.
El objetivo no se limita al procesamiento de datos, sino que prioriza su interpretación desde la lógica del proceso, considerando balances energéticos, consistencia física y desempeño operacional.

## Se implementan metodologías para:

- Validación y depuración de datos industriales
- Cálculo de indicadores térmicos clave (KPIs)
- Evaluación estadística de estabilidad operacional
- Visualización técnica orientada a toma de decisiones

El enfoque combina ingeniería de procesos con análisis cuantitativo, asegurando coherencia entre los resultados computacionales y la realidad física del sistema.

## Objetivos
- Evaluar la estabilidad operacional del sistema bajo condiciones variables.
- Detectar desviaciones térmicas respecto al comportamiento esperado.
- Cuantificar eficiencia energética mediante indicadores técnicos.
- Identificar oportunidades de mejora operacional basadas en evidencia cuantitativa.
- Integrar fundamentos de balance energético con herramientas computacionales reproducibles.

## Enfoque Metodológico
El análisis se basa en principios de ingeniería química aplicados a sistemas térmicos en régimen continuo, considerando:

- Régimen cuasi-estacionario por intervalo de medición.
- Aplicación del balance de energía como herramienta de validación.
- Supuestos termofísicos razonables (propiedades constantes).
- Evaluación estadística para caracterizar variabilidad y estabilidad.

Python se utiliza como herramienta de soporte ingenieril, priorizando claridad estructural, trazabilidad del modelo y escalabilidad hacia aplicaciones industriales más complejas (SPC, modelos predictivos, dashboards operacionales).

## Estructura del proyecto
data/      → Base de datos operacionales del proceso
src/       → Módulos de limpieza, cálculo y análisis
main.py    → Script de ejecución integral

La arquitectura modular facilita la extensión hacia:

- Integración con bases de datos industriales reales
- Implementación de control estadístico de procesos
- Desarrollo de sistemas de monitoreo continuo
- Modelos predictivos para optimización energética


## Herramientas Utilizadas

- Python 3
- pandas/       → Manipulación y estructuración de datos)
- numpy/        → Cálculo numérico y operaciones vectorizadas
- matplotlib /  → Visualización técnica de desempeño operacional


Este proyecto representa una aplicación práctica del análisis de datos industriales desde la perspectiva de la ingeniería de procesos, donde el objetivo principal es transformar información operacional en conocimiento técnico accionable.
