
# Análisis y Optimización Operacional de un Intercambiador de Calor de Doble Tubo

**Ingeniero Civil Químico | Python · Operaciones Unitarias · Análisis y Modelación de Procesos**

---

## Descripción
Este proyecto aborda el **análisis térmico y la evaluación operacional** de un intercambiador de calor de doble tubo operando en **contracorriente**, utilizando Python como herramienta de apoyo al análisis ingenieril.

El objetivo es estudiar el desempeño del equipo bajo distintas condiciones de caudal, identificar tendencias relevantes y **apoyar decisiones de diseño preliminar y operación**, priorizando la interpretación técnica de los resultados por sobre el desarrollo teórico extensivo.

---

## Objetivo
Desarrollar un modelo simplificado pero representativo que permita:

- Estimar la **carga térmica transferida** entre las corrientes
- Calcular las **temperaturas de salida** del fluido caliente y frío
- Analizar el impacto del **caudal másico** en la eficiencia térmica del sistema
- Utilizar visualizaciones para **apoyar decisiones de ingeniería de procesos**

---

## Supuestos Operacionales
El análisis se realiza bajo los siguientes supuestos habituales en evaluaciones preliminares:

- Régimen estacionario  
- Operación sin cambio de fase  
- Propiedades termofísicas constantes  
- Flujo a contracorriente  
- Pérdidas de calor al ambiente despreciables  

---

## Operación Unitaria
**Transferencia de Calor**  
Equipo analizado: **Intercambiador de calor de doble tubo**

---

## Enfoque Computacional
El modelo se implementa en Python con énfasis en:

- Código claro, estructurado y reproducible  
- Uso de análisis paramétrico para evaluar condiciones de operación  
- Visualización de resultados orientada a la **toma de decisiones técnicas**  

Python se utiliza como una **herramienta de ingeniería**, integrada al análisis del proceso.
