
# Análisis y Optimización de un Intercambiador de Calor de Doble Tubo

**Ingeniero Civil Químico | Python · Operaciones Unitarias · Modelación**

## Objetivo
Modelar y analizar un intercambiador de calor de doble tubo en contracorriente utilizando Python, con el fin de:
- Calcular carga térmica
- Estimar temperaturas de salida
- Analizar el efecto del caudal en la eficiencia térmica
- Apoyar decisiones de diseño preliminar

## Supuestos
- Régimen estacionario
- Sin cambio de fase
- Propiedades constantes
- Flujo a contracorriente

## Operación Unitaria
Transferencia de calor

## Archivos
- `intercambiador_doble_tubo.py`: script principal del análisis
- `README.md`: descripción del proyecto

## Requisitos
- Python 3.x
- numpy
- matplotlib
